Simple commands for git commit

  1. git add + (file name)
  2. git commit
  3. git push
    4. 1. To add to staging area (you can check with git status) 2. Opens new window in VS Code, and you enter the subject and body for the commit (description). Then ctrl + S , and close window. 3. To sincronize with Git Hub

    File.lock / can't commit

    When running git bash and the vs code terminal at the same time, files where duplicated and the file couldn't be commited. To solve this, run: rm -f .git/index.lock
